2.1.1 Auto-run

Auto-run is a feature that allows your character to run in a forward direction, without touching any of the arrow keys. You can turn left or right while using autorun. To toggle this on, press the [NumLock] key on the numpad. To disable this, press the same key, or the up or down arrows on the keyboard.
