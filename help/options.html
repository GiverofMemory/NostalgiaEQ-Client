4.1.6 Options and Settings

/fastdrop ON/OFF/NEVER - This toggles the ability to drop items on the ground without a confirmation box. The recommended setting for this option is the same as the default setting, which is /fastdrop off. If this is set to never, you will never be able to drop items on the ground. If this is set to on, you will always drop the item to the ground without a confirmation box.

/autojoin channel-list - This saves the channels listed, and automatically sets you to join those channels when you log in.

/chatfontsize 1-5 - This changes the size of the chat font.

/filter - This command will display a list of text filters to toggle on and off.

/inspect ON/OFF - This toggles the ability to inspect others with a right click of the mouse. Use this if you use the mouse for movement, or find yourself accidentally inspecting others often.

/log ON/OFF - This toggles the ability to log all text locally to the user's machine. The file that is generated can be found in your EverQuest directory. The format of the filename is eqlog_(server #)_(character name).txt.

/lootnodrop ALWAYS/SOMETIMES/NEVER - This command specifies how you loot no drop items on corpses. "Always" is default and will cause a prompt box to always be shown when looting no drop items. Setting this to "sometimes" causes only items not usable by your character to show a prompt box. When you have this set to "never", you will never get a confirmation box choose whether or not to loot.

/mousespeed 0-3 - This changes the mouse travel speed across the monitor. 3 is the maximum setting (the higher the number the faster the mouse moves). Don't set this above 3. Your mouse will move too quickly and you will probably not be able to play.

/shownames ON/OFF - This command turns the visible names above PC's and NPC's heads on or off.

/showspelleffects ON/OFF - Toggles spell effects on or off.

/resetwindows - Used to reset to default the locations on-screen for the different Graphical User Interface (GUI) boxes.
