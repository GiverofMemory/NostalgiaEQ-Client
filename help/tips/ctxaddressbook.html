Address Book Tips

The address book is your personal contact list. Each entry has a case-insensitive nickname that you use to identify one or more character addresses.

- To change the content of your address book you just need to fill in the name, addresses and note fields. The main button used to insert the information in the address book will be labeled "Add" if the name is not already in the address book, "Replace" if the name exists but is not the currently selected entry, or "Modify" if the currently selected entry is being modified.
- You can flag the entry name as private to prevent it from being sent to the recipients. Just before sending the message, the addresses will be expanded individually instead of being sent as part of a named list.
- The Guild mailing list is read-only and automatically kept up-to-date for you. You can however make a copy of it by selecting the Guild entry and changing its name. After you press the "Add" button, you can edit the list of addresses and the note.
- You can add a recipient to the "to" field of the mail composition window. If the currently selected entry has not been modified, you can press the "Mail To" button and the name will be inserted at the end of the list of recipients. You can also secretly add a recipient by first pressing the left ALT key, and the button will be labeled "Secret To".
- The address book is saved in a client file. The notes are never sent to the server and are completely private. Your list names are public unless you mark them as private. By default, all characters shared one address book stored in your EverQuest user data directory. You can change which address book to use when playing the current character by going to the mail options.

For more information, please see Help.
