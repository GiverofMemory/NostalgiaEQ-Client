Bandolier Tips

To add a weapon set to the bandolier:
- First, select the slot that you want to add the set to in the list window.
- Second, click the add button.
- Finally, enter a name for the set and click ok. This will add a weapon set made up of your currently equipped weapons.

To clear a weapon set in the bandolier:
- Select the slot in the list that you want to remove and click the delete button.

To activate a weapon set:
- Click the slot you want to activate then click the activate button.
- Make a hot key using the Hot Button, place the newly created hot button in the hot buttons window, and then click the button.
- Use the /bandolier activate command.
- Check the auto swap check box and then anytime you select a set from the list it will be activated.

The items displayed on the left side of the bandolier window are representations of the items in the selected weapon set. If you no longer have an item in your inventory the background for that item display will be red.
When you activate a weapon set the currently wielded items will be auto-inventoried into your general inventory.
