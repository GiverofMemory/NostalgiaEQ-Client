Task Selection Tips

This window displays a list of Tasks being offered to you.

- To display a Task's description, select the Task in the Task List. The Task's description should show in the window below the list.

- To accept a Task, select the Task in the Task List and click the Accept button.

- To decline the Tasks, click the Decline All button, and you will not be assigned any of the Tasks. Note that you may only request new tasks every 15 seconds.

For more information, please see Help.
