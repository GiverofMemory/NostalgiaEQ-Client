Voice Macro Tips

This window can be used to send Voice Macros, as well as create hotbuttons that will send Voice Macros when pressed.

- To hear a macro, select the macro in the list and press Test.

- To send a macro to one person, target that person, select a macro and click Tell.

- To send a macro to your group or raid, select a macro and click the appropriate button (Group or Raid).

- To create a hotbutton, select a macro and click-and-hold the appropriate button. The hotbutton created, once placed in the HotBar, can be edited to change the macro select, change targets, etc.

- To play/send macros from the chat line, use one of these commands: /vplay, /vtell, /vgroup, /vraid. Type /vplay alone on the chatline to see a Usage statement.

For more information, please see Help.
