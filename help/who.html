3.7 The "/who" Command

This is a very useful command that allows you to see who is online within the search parameters. Simply typing /who in the chat bar and hitting enter will bring up a list of everyone currently in the zone you are in. There is an additional argument that allows you to run a search worldwide, this is the /who all command. Listed below are a few examples of the /who command in action.

* /who all necromancer - Lists all necromancers in the world.

* /who all 5 - Lists all the level 5 characters in the world.

* /who all cleric 3 - Lists all the level 3 clerics in the world.

* /who all Maddoc - This returns any character whose name begins with "Maddoc". If no such person is online, it will return with a "There are no players in EverQuest who match those filters."

** NOTE: If there are too many matches to the query, it will cut the list short; you may need to refine the query a bit.
